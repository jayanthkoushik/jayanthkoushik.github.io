Projects

Understanding Convolutional Neural Networks

This was my project for the Statistical Machine Learning course at CMU. My goal was to gain a better mathematical understanding of why convolutional neural networks work. Stéphane Mallat has pioneered a mathematical framework (summarized in this paper) for analyzing CNN properties. The framework is based on wavelet scattering, and the focus of my project was understanding its key results. My project report provides a higher level introduction to this work, and the PhD thesis of Mallat's student Joan Bruna is another valuable source of information on this topic. Bruna also teaches a relevant course at UC Berkeley - the slides are on GitHub!

Learning Graph Matching with Deep Neural Networks

Graph matching is the problem of finding a correspondence between nodes of two graphs. It is a fundamental problem in pattern recognition. Graph matching itself is not a machine learning task - it is typically framed as a linear or quadratic assignment problem with a fixed matrix that specifies the compatibility between vertices and / or edges of the two graphs. However, Caetano et al. showed that by learning compatibility matrices specific to a task, it is possible to obtain much better results. In this project, done for the Machine Learning course at CMU, we tried to extend this work by using deep neural networks as the learning algorithm, and tried different architectures and loss functions. Unfortunately, we did not get any positive results.

One of my teammates, Alfred Nguyen, followed up with this project and was able to get very promising results with a metric learning approach!
